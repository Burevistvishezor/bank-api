# Bank API (FastAPI + PostgreSQL)

Современный RESTful API для банковских симуляций, построенный на высокопроизводительном стеке.

## 🚀 Стек технологий
* **FastAPI** — асинхронный фреймворк для создания API.
* **PostgreSQL** — надежная реляционная база данных.
* **SQLAlchemy** — мощная ORM для работы с данными.
* **Pydantic** — валидация данных и работа со схемами.
* **Alembic** — миграции базы данных.

## 🛠 Функционал
- [x] Создание и управление пользователями.
- [x] Открытие мультивалютных счетов.
- [x] Проведение транзакций между счетами.
- [x] Валидация остатка перед списанием.

## 📦 Быстрый запуск (Docker)
```bash
# Клонируйте репозиторий
git clone github.com

# Запустите контейнеры
docker-compose up -d
```
После запуска документация доступна по адресу: `http://localhost:8000/docs`
